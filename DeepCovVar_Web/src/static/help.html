DeepCovVar Tutorial

Welcome to the DeepCovVar help page. This guide will walk you through using the web interface to classify COVID-19 variants. If you have questions not covered here, please visit our GitHub repository or contact us.

Getting Started

DeepCovVar provides a user-friendly web interface for submitting sequence data and receiving variant classification results. The process involves three main steps: providing input data, selecting prediction options, and viewing results.

Data Input

Sequence Type Selection

First, select your query sequence type:

  • Amino acid (Protein): Select this if your sequences are protein sequences
  • Nucleotide: Select this if your sequences are DNA/RNA sequences

Note: DeepCovVar can automatically detect and convert nucleotide sequences to protein sequences using Prodigal integration.

Input Methods

DeepCovVar supports three convenient ways to provide sequence data:

1. Accession ID

Enter an accession ID from NCBI or UniProt databases. The system will automatically fetch the sequence for you.

  • Select NCBI for NCBI GenBank/RefSeq accessions (e.g., P0DTC2)

2. File Upload

Upload a FASTA file containing your sequences:

  • Click "Browse" to select a file from your computer
  • Supported formats: .fasta, .fa, .faa, .fna, .txt
  • Maximum: 1000 sequences per file
  • File size limit: 50 MB

3. Paste Sequences

Directly paste FASTA-formatted sequences into the text area. Click "Load Demo FASTA" to see an example format.

FASTA Format

Sequences must be in FASTA format:

>Sequence_ID Description
MFVFLVLLPLVSSQCVNLTTRTQLPPAYTNSFTRGVYYPDKVFRSSVLH
STQDLFLPFFSNVTWFHAIHVSGTNGTKRFDNPVLPFNDGVYFASTEKS
...

Prediction Phases

DeepCovVar uses a five-phase hierarchical classification system. You can select which phases to run:

Phase I: Virus vs Non-virus

The first phase determines whether the input sequence is from a virus or not. This is the foundational classification step.

Phase II: (+)ssRNA (Class IV) vs Others

For sequences classified as viruses, this phase identifies positive-sense single-stranded RNA viruses (Class IV).

Phase III: Coronavirus vs Other ssRNA(+)

This phase specifically identifies coronavirus sequences among positive-sense ssRNA viruses.

Phase IV: SARS-CoV-2, MERS, and SARS Classification

Phase IV distinguishes between SARS-CoV-2, MERS-CoV, and SARS-CoV among coronavirus sequences.

Phase V: SARS-CoV-2 Variant Classification

The final phase classifies SARS-CoV-2 sequences into specific variants:

  • Omicron
  • Alpha
  • Delta
  • Epsilon
  • Iota
  • Gamma
  • Others

Selecting Phases

  • Check individual phases you want to run
  • Click "Select All Phases" to run the complete pipeline (recommended for comprehensive analysis)
  • At least one phase must be selected

Options

Model Selection

  • Optimized Models (Recommended): Uses space-optimized models for faster processing with minimal accuracy trade-off
  • Standard Models: Uses full-size models for maximum accuracy

Verbose Output

Enable this option to receive detailed logging information about the prediction process. Useful for debugging or understanding the classification steps.

Running Predictions

Email Notification (Optional)

Enter your email address to receive a notification when your prediction job completes. This is especially useful for large datasets that may take longer to process.

Submitting Your Job

  1. Ensure you've provided input data (accession ID, file, or pasted sequences)
  2. Select at least one prediction phase
  3. Choose your preferred options
  4. Click "Run Prediction"

You'll receive a unique Job ID and be redirected to the results page where you can monitor progress.

Understanding Results

Results Page

The results page displays:

  • Job Status: Pending, Running, Completed, or Failed
  • Progress Bar: Real-time progress indicator
  • Phase Results: Classification results for each completed phase
  • Visualizations: Charts showing variant distribution and confidence scores

Downloading Results

Results can be downloaded in multiple formats:

  • CSV: Comma-separated values for easy import into spreadsheet software
  • Excel: Formatted Excel workbook with separate sheets for each phase
  • PDF: Professional report with summary statistics and results

Troubleshooting

Common Issues

Invalid FASTA Format

Problem: Error message about invalid FASTA format
Solution: Ensure your sequences follow proper FASTA format with header lines starting with ">" followed by sequence data

Too Many Sequences

Problem: Error about exceeding sequence limit
Solution: Split your dataset into smaller batches (maximum 1000 sequences per submission)

Job Taking Too Long

Problem: Job appears stuck or taking excessive time
Solution: Large datasets or running all phases can take time. Check back later or use email notification. Jobs timeout after 1 hour.

Accession ID Not Found

Problem: Error fetching sequence from database
Solution: Verify the accession ID is correct and select the appropriate database (NCBI or UniProt)

Best Practices

  • Use "Select All Phases" for comprehensive variant analysis
  • Provide email address for large jobs to receive completion notifications
  • Check sequence quality before submission to ensure accurate results
  • Use demo data to familiarize yourself with the interface
  • Keep Job IDs to retrieve results later
