CLI Tool Installation

Requirements

  • Python 3.7 or higher
  • pip package manager
  • Prodigal (for nucleotide sequence conversion)

Installation Steps

1. Clone the Repository

git clone https://github.com/ckoh04/DeepCovVar.git cd DeepCovVar

2. Install Python Dependencies

pip install -r requirements.txt

3. Install Prodigal

Option 1: Using conda

conda install -c bioconda prodigal

Option 2: From source

git clone https://github.com/hyattpd/Prodigal.git cd Prodigal make install

4. Install DeepCovVar Package

pip install .

Quick Start

Command Line Usage

Run complete pipeline (all phases):

python -m deepcovvar -f input.fasta -o output_dir --all-phases

Run specific phase:

python -m deepcovvar -f input.fasta -o output_dir -p 5

Run with verbose output:

python -m deepcovvar -f input.fasta -o output_dir --all-phases --verbose

Python API Usage

from deepcovvar.covid_classifier import COVIDClassifier # Initialize classifier classifier = COVIDClassifier() # Run complete pipeline (all phases) all_results = classifier.run_all_phases( input_file="sequences.fasta", output_dir="results/", base_filename="my_analysis" )

System Requirements

Minimum Requirements

  • OS: Linux, macOS, or Windows
  • RAM: 8 GB (16 GB recommended for large datasets)
  • Storage: 2 GB for models and dependencies
  • Python: 3.7+

Recommended Requirements

  • RAM: 16 GB or more
  • CPU: Multi-core processor (4+ cores)
  • GPU: Optional, but recommended for Phase 5 (CUDA-compatible)
